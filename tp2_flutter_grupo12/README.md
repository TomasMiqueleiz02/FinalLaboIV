# Jugadores App

Esta es una aplicación desarrollada en Flutter que permite visualizar una lista de jugadores obtenida desde una API y ver detalles individuales de cada uno. También cuenta con una opción para activar o desactivar el modo oscuro desde la pantalla de perfil.

## Características

- Lista de jugadores obtenida desde una API REST.
- Pantalla de detalles de cada jugador.
- Opción de activar o desactivar el modo oscuro.
- Navegación entre pantallas mediante un menú lateral (Drawer).
- Diseño adaptable a modo claro y oscuro.
